# MTSDR _ `MIDI To Song (for Drums Rock)`
### (Feb 2025) This program was Created with GameMaker by BubtwnMB (u/AsideBig2747) 
#### This was made with the help of Garage 51's instructions: https://garage51.es/drumsrock/custom_songs.html
***
Originally I haven't thought of sharing this program, but I decided to share it because I think y'all guys would like to use it. 

Before you can export a Custom Song, you must require: 
- the Main Audio (in the `.ogg` format)
- the Preview (in the `.ogg` format)
- and the Drum MIDI

There are multiple ways you can create Drum MIDI; from MIDI Sequencers to DAWs (Digital Audio Workstations).
It would be best to use a website or software that would allow you to include an Audio File so you easily align the Drum MIDI with the Audio. 
Personally, I use FL Studio (a DAW).

Hopefully this helps :)
